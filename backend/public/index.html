🎬 Virlo AI Backend API

● Online

Welcome to the Virlo AI video generation backend API. This service powers automated UGC video creation using AI.

🚀 API Endpoints

GET /health
Health check endpoint - returns API status and version information
POST /api/video
Generate video content using AI pipeline
POST /api/script
Generate video scripts from product URLs
POST /api/avatar
Create AI avatars for video content
POST /api/webhooks/virlo
Webhook endpoint for external service callbacks
